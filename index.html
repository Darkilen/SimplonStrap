La Barre de Navigation

Notre barre de navigation vous permettra d'acquérir un menu Hamburger responsive !

Sa construction

Un container doté de la classe .nav pour l'adaptation à n'importe quel type de menu et non uniquement une barre de navigation
On initialise ensuite une liste à puce avec la balise ul comprenant les divers éléments du menu
On ajoute un second container avec un id hamburger qui permettra le declenchement du déroulement du menu en hover
On ajoute un troisième container avec un id Navico qui engloble l'icone de l'hamburger en lui même et une largeur égale à celle de son parent pour une meilleure accessibilité du déroulement sur mobile
On crée ensuite les différents liens du menu entre les balises li de la liste à puce

Le responsive

L'adaptabilité du menu est geré par des media queries à definir en fonction de votre design personel
- Sur Desktop :
Par défaut, nous avons fait le choix de cacher l'icone Hamburger. Les éléments de la liste sont sous la forme d'inline-block.
- Sur Mobile :
Les éléments des listes sont cette fois ci affiché en tant que block et restent cliquables sur la totalité de la largeur de l'ecran

La Grille

La grille permet deplacer et de dimensionner facilement ses divs et ses colonnes. "containflex" permet d'initialiser la grille. La taille de la grille prend celle de la div sur laquelle elle est initialisée. "colnorm" créer une colonne qui adapte ça taille aux nombres de colonnes placer dans le "containflex". "colnorm" donne donc la même taille à toutes les divs. Vous pouvez aussi choisir des tailles : petit, moyen ou grand avec "coltiny", "colmedium" et "colbig". Vous disposez aussi de colonne "flex" qui permette de mettre des colonnes dans des colonnes car elles font office de container.

Le Html

<div style=" height: 600px; border: 1px solid black;" class="containflex"> <div style="background-color: red; padding: 16px;" class="colflexnorm"> <div style="background-color: blue; margin: 6px;" class="coltiny"> </div> <div style="background-color: blue; margin: 6px;" class="colmedium"> </div> <div style="background-color: blue; margin: 6px;" class="colbig"> </div> </div> <div style="background-color: blue;" class="colnorm"> </div> </div>

Le Css

/* -----------------------------------------------Grid */ .containflex { display: flex; box-sizing: border-box; } .colnorm { flex: 1; box-sizing: border-box; } .coltiny { flex: 0.5; box-sizing: border-box; } .colmedium { flex: 2; box-sizing: border-box; } .colbig { flex: 4; box-sizing: border-box; } .colflexnorm { display: flex; flex: 1; box-sizing: border-box; } .colflextiny { display: flex; flex: 0.5; box-sizing: border-box; } .colflexmedium { display: flex; flex: 2; box-sizing: border-box; } .colflexbig { display: flex; flex: 4; box-sizing: border-box; }

Le Formulaire

Voici donc le formulaire, très simple et modulable à souhait. Vous remarquerez le design flat afin d'améliorer la lisibilité.

Pour obtenir ce formulaire pré-formaté, il suffit d'attribuer la classe "form" à votre formulaire.

La balise form n'a pas de bordure mais un border radius afin d'alléger le design, un léger box-shadow a été ajouté afin de bien le distinguer dans le DOM.

Les Fieldset ont une bordure grise toute simple, un text-align:center est appliqué afin de centrer son contenu.

Les legend sont en italique avec un text-align: left.

Les label sont en inline-block afin de pouvoir utiliser un text-align:left tout en leur attribuant une width.
Vous remarquerez qu'ils sont également en bold afin d'améliorer leur lisibilité.

Les imput ont un padding de 5px, toujours pour améliorer la lisibilité. Leur width est fixée à 40% du parent.
Vous noterez que les input de type radio et checkbox ont une width:auto afin d'etre au plus près de leurs labels respectifs.

Une transition de 0.5s a été ajoutée sur le focus des input, toujours afin de rentre le visuel plus agréable.
Les balises textarea sont quant à elles d'une width de 40% afin d'etre en accord avec les input.

La balise select à été egalement remodelée très légèrement avec une width de 40% (toujours pour la même raison) et un border de la même couleur que les input.

Enfin concernant le button, la bordure a été enlevée, un background-color vert ajouté, une petite box shadow appliquée de manière très légère et la couleur du texte changé en blanc.
Lorsque ce bouton est activé, un box shadow en inset est appliqué afin de donner un effet de bouton qui "s'enfonce".

Le Tableau

Ajoutez simplement la classe "table" a votre balise table, les effet sont décrit dans le tableau d'exemple suivant.

Attribut Valeur Description
.table[margin] auto Centre le tableau
.table[background] #EEEEEE Couleur grise de background
.table[border-collapse] separate Sépare les borders afin d'améliorer la lisibilité
.table[font-size] 0.9em Diminue un peu la taille du texte
.table[width] 80% Définie la largeur à 80%
.table[text-align] left Aligne le texte à gauche
.table[box-shadow] 0 0 10px 0 rgba(0, 0, 0, .3) Applique une ombre très légère tout autour du tableau
th[font-weight] bold Met la police des cellules du head du tableau en gras
th[padding] 10px 15px Padding améliorant la lisibilité
th[color] #fff Couleur de police blanche
th[background] linear-gradient(#777, #444) Background en dégradé
td[border] 1px solid #fff Bordure blanche
tbody tr td:first-child[font-weight] bold Met en gras la première cellule de chaque ligne

Les Boutons

Ci-dessous vous trouverez les boutons:

Un bouton neutre, attribuez la class .btn, avec un border gris, un background en radial-gradient gris très léger, une width de 100px, un padding vertical de 8px afin d'ameliorer la lisibilité, un cursor: pointer,
une suppression de l'outline lors d'un focus et enfin un shadow inset afin de donner une impression "d'enfoncer" le bouton quand il est activé.

Ajoutez la classe "vld" afin de changer la couleur de background en vert.

Ajoutez la classe "anl" afin de changer la couleur de background en rouge.

Le HTML

<button class="btn" type="button" name="button">Cliquez ici</button> <button class="btn vld" type="button" name="button">Valider</button> <button class="btn anl" type="button" name="button">Annuler</button>

Le Css

.btn { padding: 8px 0px; width: 100px; border: 1px solid #CCCCCC; border-radius: 5px; background: radial-gradient(#EEEEEE, #CCCCCC); cursor: pointer; } .btn.vld { background:radial-gradient(#7FFF00, #32CD32); } .btn.anl { background:radial-gradient(#FF0000, #CF0A1D); } .btn:active { box-shadow: inset 1px 1px 5px grey; } .btn:focus { outline: none; }

Le Reset User Agent

Le Reset User Agent permet de réinitialiser les paramètres d'affichage par défaut du navigateur que l'on ne souhaite pas conserver dans notre code

Le Css

/* -----------------------------------------------Reset User Agent */ * { margin: 0; /* Reintialise all margin*/ padding: 0; /* Reintialise all padding*/ list-style-type: none; /* Delete Bullet on list */ text-decoration: none; /* Delete Underline on Link */ box-sizing: border-box; /* Padding and Border include in box size */ } img { max-width: 100%; }

La Barre de Navigation

La nav est la même que celle du menu de ce site.

Sa construction

Un container doté de la classe .nav pour l'adaptation à n'importe quel type de menu et non uniquement une barre de navigation
On initialise ensuite une liste à puce avec la balise ul comprenant les divers éléments du menu
On crée ensuite les différents liens du menu entre les balises li de la liste à puce
Pour intégrer un sous-menu, utilisez la classe "hid_ul" sur l'ul du sous-menu.

Le responsive

L'adaptabilité du menu est geré par des media queries à definir en fonction de votre design personel
- Sur Desktop :
Les éléments de la liste sont en flex 1.
- Sur Mobile :
L'affichage est en block sur un click de "Menu", les sous-menus sont affichés mais avec un font-size inférieur et des bordures afin de bien les distinguer

Le Html

<nav class="nav" role="navigation"> <ul> <li><a href="#">Accueil</a></li> <li><a href="#menu">Navigation</a></li> <li><a href="#grid">Grille</a> <ul class="hid_ul"> <li><a href="#">One</a></li> <li><a href="#">Two</a></li> <li><a href="#">Three</a></li> </ul> </li> <li><a href="#form">Formulaire</a></li> <li><a href="#tableau">Tableau</a></li> <li><a href="#buttons">Boutons</a></li> <li><a href="#reset">Reset</a></li> <li><a href="#helper">Helper Class</a></li> <li><a href="#contact">Contact</a></li> </ul> </nav>

La Grille Flex

La grille permet de placer et de dimensionner facilement ses divs et ses colonnes. "containflex" permet d'initialiser la grille. La taille de la grille prend celle de la div sur laquelle elle est initialisée. "colnorm" crée une colonne qui adapte sa taille aux nombres de colonnes placées dans le "containflex". "colnorm" donne donc la même taille à toutes les divs. Vous pouvez aussi choisir des tailles : petit, moyen ou grand avec "coltiny", "colmedium" et "colbig". Vous disposez aussi de colonne "flex" qui permette de mettre des colonnes dans des colonnes car elles font office de container.

Le Html

<div style="border: 1px solid black;" class="containflex h600p"> <div style="background-color: red; padding: 16px;" class="colflexnorm"> <div style="background-color: blue; margin: 6px;" class="coltiny"> </div> <div style="background-color: blue; margin: 6px;" class="colmedium"> </div> <div style="background-color: blue; margin: 6px;" class="colbig"> </div> </div> <div style="background-color: blue;" class="colnorm"> </div> </div>

Le Css

/* -----------------------------------------------Grid */ .containflex { display: flex; box-sizing: border-box; } .colnorm { flex: 1; box-sizing: border-box; } .coltiny { flex: 0.5; box-sizing: border-box; } .colmedium { flex: 2; box-sizing: border-box; } .colbig { flex: 4; box-sizing: border-box; } .colflexnorm { display: flex; flex: 1; box-sizing: border-box; } .colflextiny { display: flex; flex: 0.5; box-sizing: border-box; } .colflexmedium { display: flex; flex: 2; box-sizing: border-box; } .colflexbig { display: flex; flex: 4; box-sizing: border-box; }

La Grille sans Flex

Pour les amateurs de BootStrap et les vétérans d'Internet Explorer, nous avons également implémenté une grille de base 12 colonnes, n'utilisant aucun flex.
classe du container: .grid
Fonctionnement des dimensions des colonnes: w-dividende-diviseur, allant de 1 à 12
Exemples :

  • une colonne de taille un douzième: "w-1-12" = "one-twelve"
  • une colonne de taille un tier: "w-1-3" = "w-4-12" = "four-twelve"
  • une colonne faisant la moitié de la taille de son parent: "w-1-2" = "w-2-4" = "w-3-6" = "w-6-12" = "six-twelve"

Pour créer vos propres grilles, je vous propose d'utiliser le PGCD de leurs noms de classe. Par exemple, si vous souhaitez y mettre deux colonnes - disons un tier et deux tiers - vous pourriez leur ajouter les classes "w-1-3" et "w-2-3"; par contre, si vous souhaitez ajouter une colonne dont le PGCD est 12, mon conseil est d'utiliser uniquement les classes de largeur de colonnes terminant par 12 ("w-1-12", "w-2-12", ..., "w-12-12")

Le Formulaire

Voici donc le formulaire, très simple et modulable à souhait. Vous remarquerez le design flat afin d'améliorer la lisibilité.

Pour obtenir ce formulaire pré-formaté, il suffit d'attribuer la classe "form" à votre formulaire.

La balise form n'a pas de bordure mais un border radius afin d'alléger le design, un léger box-shadow a été ajouté afin de bien le distinguer dans le DOM.

Les Fieldset ont une bordure grise toute simple, un text-align:center est appliqué afin de centrer son contenu.

Les legend sont en italique avec un text-align: left.

Les label sont en inline-block afin de pouvoir utiliser un text-align:left tout en leur attribuant une width.
Vous remarquerez qu'ils sont également en bold afin d'améliorer leur lisibilité.

Les imput ont un padding de 5px, toujours pour améliorer la lisibilité. Leur width est fixée à 40% du parent.
Vous noterez que les input de type radio et checkbox ont une width:auto afin d'etre au plus près de leurs labels respectifs.

Une transition de 0.5s a été ajoutée sur le focus des input, toujours afin de rentre le visuel plus agréable.
Les balises textarea sont quant à elles d'une width de 40% afin d'etre en accord avec les input.

La balise select à été egalement remodelée très légèrement avec une width de 40% (toujours pour la même raison) et un border de la même couleur que les input.

Enfin concernant le button, la bordure a été enlevée, un background-color vert ajouté, une petite box shadow appliquée de manière très légère et la couleur du texte changé en blanc.
Lorsque ce bouton est activé, un box shadow en inset est appliqué afin de donner un effet de bouton qui "s'enfonce".

Les Boutons

Ci-dessous vous trouverez les boutons:

Un bouton neutre, attribuez la class .btn, avec un border gris, un background en radial-gradient gris très léger, une width de 100px, un padding vertical de 8px afin d'ameliorer la lisibilité, un cursor: pointer,
une suppression de l'outline lors d'un focus et enfin un shadow inset afin de donner une impression "d'enfoncer" le bouton quand il est activé.

Ajoutez la classe "vld" afin de changer la couleur de background en vert.

Ajoutez la classe "anl" afin de changer la couleur de background en rouge.

Le HTML

<button class="btn" type="button" name="button">Cliquez ici</button> <button class="btn vld" type="button" name="button">Valider</button> <button class="btn anl" type="button" name="button">Annuler</button>
